<img src="https://media.giphy.com/media/l0HlNaQ6gWfllcjDO/giphy.gif" width="300"/>

- ✈️ Airline performance patterns  
- ⏱️ Delay trends  
- ❌ Cancellation behavior  
- 🏢 Airport traffic insights
