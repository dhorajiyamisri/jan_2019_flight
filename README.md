# ✈️ Airline Flight Delay & Cancellation Analytics

![Banner](images/banner.png)

---

## 📌 Project Overview

This project analyzes airline flight delays and cancellations using **Python** and **Power BI**. The objective is to identify operational patterns, evaluate airline performance, and build an interactive dashboard for business insights.

---

## 🎯 Project Objectives

- Analyze flight delay patterns
- Analyze cancellation trends
- Compare airline performance
- Identify busiest airports
- Build an interactive Power BI dashboard
- Generate business insights

---
