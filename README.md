# 📌 ABOUT PROJECT

This project analyzes airline **flight delays and cancellations** using Python and Power BI.

✔ Identify delay patterns  
✔ Understand cancellation trends  
✔ Compare airline performance  
✔ Build interactive dashboard  

---

# ✨ FEATURES

- ⚡ Data cleaning & preprocessing  
- 📊 Exploratory Data Analysis (EDA)  
- 📉 Delay & cancellation analysis  
- 📈 Visual insights using charts  
- 📊 Power BI dashboard integration  

---

# 📂 DATASET INFORMATION

| Column | Meaning |
|--------|--------|
| OP_CARRIER | Airline Code |
| ORIGIN | Departure Airport |
| DEST | Arrival Airport |
| DEP_DEL15 | Departure Delay |
| ARR_DEL15 | Arrival Delay |
| CANCELLED | Flight Cancel Status |
| DISTANCE | Flight Distance |
