## 📌 About Project

✈️ This project analyzes airline **flight delays & cancellations** using Python and Power BI.

✔ Understand delay patterns  
✔ Analyze cancellations  
✔ Compare airline performance  
✔ Generate business insights  

---

## 🎯 Project Objectives

- Analyze departure & arrival delays  
- Study cancellation trends  
- Identify busiest airports  
- Compare airline performance  
- Support data-driven decisions  

---

## 📂 Dataset

| Feature | Description |
|--------|-------------|
| OP_CARRIER | Airline |
| ORIGIN | Departure Airport |
| DEST | Arrival Airport |
| DEP_DEL15 | Departure Delay |
| ARR_DEL15 | Arrival Delay |
| CANCELLED | Flight Status |
